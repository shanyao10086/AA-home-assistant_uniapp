 用户交互层：微信小程序
    角色：      最终用户的操作界面。
    核心功能：
    设备管理：  扫描、添加、删除 ESP32 边缘设备。
    网络配置：  引导用户为 ESP32 配置 Wi-Fi（SSID/密码）。
    网关配置：  让用户输入或自动发现树莓派中心网关的 IP 地址和端口。
    状态展示：  实时显示从网关获取的温湿度数据。
    远程控制：  发送“开启/关闭空调”等指令。
    关键技术：

    配网方案：  
                使用 ESP-IDF 的 Wi-Fi Provisioning 功能。小程序通过蓝牙或 SoftAP 模式与 ESP32 临时建立连接，安全地传输 Wi-Fi 凭据和网关地址。

    与服务层通信：
                通过 HTTPS/WebSocket 与你的本地测试服务器（未来可迁移到云服务器）进行交互。
